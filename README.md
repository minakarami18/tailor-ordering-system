# tailor-ordering-system
دوزینو — سامانه سفارش‌گیری خیاطی

معرفی

این پروژه یک سامانه سفارش‌گیری آنلاین برای خیاطی است که به مشتری این امکان را می‌دهد تا طرح لباس را انتخاب کرده، گزینه‌هایی مانند جنس پارچه، رنگ، سایز و آپشن‌های اضافی را مشخص کند و سپس تخمین بازه قیمت و تخمین زمان تحویل دریافت کرده و سفارش خود را ثبت و پیگیری کند.

اهداف پروژه

* نظم‌دهی به فرآیند سفارش‌گیری (به‌جای پیام‌های پراکنده در شبکه‌های اجتماعی)
* کاهش خطا در ثبت جزئیات سفارش (سایز/رنگ/پارچه/آپشن‌ها)
* شفاف‌سازی بازه قیمت و زمان تحویل برای مشتری
* امکان مدیریت سفارش‌ها برای خیاط یا ادمین و پیگیری وضعیت سفارش توسط مشتری

 امکانات اصلی 

* نمایش کاتالوگ طرح‌ها (عکس + توضیح)
* صفحه جزئیات طرح و انتخاب گزینه‌ها (پارچه/رنگ/سایز/آپشن‌ها)
* محاسبه و نمایش تخمین بازه قیمت و تخمین زمان تحویل
* ثبت سفارش و تولید شماره سفارش
* پیگیری وضعیت سفارش (در حال بررسی / در حال دوخت / آماده / تحویل)
* پنل ادمین ساده برای مدیریت سفارش‌ها (مشاهده، تایید، تغییر وضعیت)

نقش‌ها

*مشتری: مشاهده طرح‌ها، انتخاب گزینه‌ها، ثبت سفارش، پیگیری سفارش
* ادمین یا خیاط: مدیریت طرح‌ها و سفارش‌ها، تایید/اصلاح قیمت و زمان، تغییر وضعیت سفارش

مستندات پروژه

* صورت‌جلسه‌ها: `docs/meetings/`


## ابزارها و مدیریت نسخه

* GitHub برای مدیریت نسخه و همکاری تیمی
* هر تغییر از طریق **Branch → Pull Request → Merge** انجام می‌شود.
* شاخه `main` محافظت‌شده است.
